## Awesome Chat
Chat desenvolvido com Socket.io e Express.JS.

### Pré Requisitos
- [NPM/Node](https://nodejs.org/en/download/)

### Construido com
- EJS
- Express
- Jquery
- Socket.io

### Instalação
Faça a instalação das dependências de desenvolvimento
```sh
npm install
```

Rode a aplicação localmente
```sh
npm start
```
